## Dependencies

1. React.js
2. Typescript
3. UmiJS
4. Ant design
5. Bip39
6. bitcoin.js
7. Jest
8. ahooks

## Demo page

https://kencckw.github.io/bitcoin-playground/

## Getting Started

#### Install dependencies

```bash
$ yarn
```

#### Start the dev server
```bash
$ yarn start
```

#### Running tests
```bash
$ yarn test
```

#### Security

1. disable autocomplete  
2. Password field for sensitive data
3. dependentbot
